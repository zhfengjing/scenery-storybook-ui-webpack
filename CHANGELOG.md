# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-28

### Added

- Initial release
- Button component with multiple variants (default, destructive, outline, ghost, link)
- Button component with multiple sizes (sm, default, lg, icon)
- Dialog component built with Radix UI
- TailwindCSS integration for styling
- TypeScript support
- Storybook documentation
- Webpack build configuration
- NPM publish configuration
- GitHub Pages deployment for Storybook
